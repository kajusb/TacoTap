# Android Lab 1 - TacoTap

Submitted by: **Kajus Buikus**

**TacoTap** is an Android clicker game where users tap a taco to earn tacos and unlock upgrades that infinitely increase the number of tacos earned per click.

Time spent: **0.5** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **User can see a number displayed on the screen. The number starts at 0.**
- [x] **User can tap on a button to see the number displayed increase by 1.**

The following **optional** features are implemented:

- [x] User has increasingly difficult goals to reach in terms of the number of taps accumulated.
- [x] Custom background and button colors (red and green theme)

The following **additional** features are implemented:

* [x] ImageButton used as the main tap interaction
* [x] Dynamic upgrade text showing cost and taco increase
* [x] Formatted numbers to prevent long decimal values
